Gestor documental OCR con IA local

Aplicación de escritorio para Windows orientada a automatizar el procesamiento de documentos técnicos. Permite cargar PDFs e imágenes, extraer texto con OCR, generar metadatos, guardar resultados en SQLite y apoyar la extracción de información mediante IA local con Ollama.

Python Tkinter SQLite PyMuPDF Tesseract OCR Ollama qwen2.5:7b
Ver repositorio Volver a proyectos

Enfoque

  • Automatización documental para reducir tareas manuales de revisión y clasificación.
  • Procesamiento local de PDFs e imágenes con extracción de texto mediante OCR.
  • Uso de IA local para asistir la extracción de datos sin depender de servicios externos.
  • Persistencia en SQLite para conservar textos, metadatos y resultados procesados.

Qué implementa

  • Interfaz de escritorio con Tkinter para gestionar el flujo de trabajo.
  • Lectura de PDFs con PyMuPDF y tratamiento de imágenes con Pillow.
  • Extracción OCR mediante Tesseract para documentos escaneados o imágenes.
  • Integración con Ollama y el modelo qwen2.5:7b para extracción asistida.

Estado

  • Prototipo funcional preparado para portfolio.
  • Proyecto orientado a validar un flujo real de OCR, metadatos e IA local.
  • No hay demo online asociada; el código fuente está disponible en GitHub.

Tecnologías

  • Python, Tkinter y SQLite para aplicación local y persistencia.
  • PyMuPDF, Tesseract OCR y Pillow para procesamiento documental.
  • Ollama con qwen2.5:7b para asistencia con IA ejecutada en local.
